                    function reverseString(inputString) {

                        let result = '';

                        for(let index = inputString.length - 1; index >= 0; index-- ) {
                            // This will make it into an Array
                            // result.push(inputString[index]);

                            result += inputString[index];
                        }


                        return result;
                    }

                    function displayString(stringResult) {
                        document.getElementById('alert').classList.remove('invisible');
                        document.getElementById('msg').innerHTML = `Your string reversed is: <strong>${stringResult}</strong>`;


                    }

The code is structured in three functions.

getValues()

The getValues function gets the string from the user and than sends the string to the reverseString function. The result of this string is sent to the displayString function.

reverseString()

The reverseString function reverses the inputed string. This is done by using a for loop to go backward through the string and place them into a new string. which is sent back to the getValues function.

displayString()

The displayString function gets the reversed string from the getValues function. This than makes the alert on the page visible by getting it by Id with the string that is sent bolded to show what was sent.
